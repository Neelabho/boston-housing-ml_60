About this project

This web app predicts median home values in Boston neighborhoods using the 1978 Harrison & Rubinfeld dataset.
(Dataset: https://www.openml.org/search?type=data&status=active&id=531&sort=runs).
A Gradient Boosting Regressor trained on 13 socioeconomic and geographic features achieves an R² of approximately 0.89 on held-out test data.

The model was trained using scikit-learn, saved using joblib, and served through a Flask web server. The frontend communicates with the backend via asynchronous JavaScript fetch requests so no page reloads occur.
